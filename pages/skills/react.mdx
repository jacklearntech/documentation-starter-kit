## React
免费且系统的入门教程看这里： https://www.codecademy.com/courses/react-101/lessons/react-jsx-intro/exercises/hello-world

如果要最新的，毕竟React每年都有新的变化，可以直接YouTube搜索React Course然后按照火热程度倒叙。

接触React的契机是在接手了n个不同项目后，发现很多时候需要重复开发使用类似的模块，比如带标签切换的并列三个圆环图。
当时还在用Angular，当然也有很多方法构建可以重复使用的module，但是总觉得不如React那么直接，那么方便，真正能做到复制黏贴就能搞定。
